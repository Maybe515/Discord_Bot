サーバーを監視するDiscord bot  
コマンドでサーバープロセスを実行・停止したり、各サーバーの入退室（join, Left）を表示させるbot
<br>

## 開発環境
- Windows10
- Python 3.8.2
- discord.py 1.3.3

## コマンド一覧
- /mcstart　　Minecraftのサーバープロセスを実行する
- /mcstop　　Minecraftのサーバープロセスを終了させる
- /ckstart　　Core Keeperのサーバープロセスを実行する
- /ckstop　　Core Keeperのサーバープロセスを実行する
- /help　　　コマンド一覧を表示する
